setpci

查询和配置PCI设备的使用工具

补充说明

setpci命令 是一个查询和配置PCI设备的使用工具。

语法

选项

参数

  • PCI设备：指定要配置的PCI设备；
  • 操作：指定要完成的配置操作。

实例

首先进入终端输入lspci命令，列出各种设备的地址：

发现00:02.0是VGA设备，于是我们修改它的属性：

解释一下：

  • setpci 是修改设备属性的命令。
  • -s 表示接下来输入的是设备的地址。
  • 00:02.0 VGA设备地址（<总线>:<接口>.<功能>）。
  • F4 要修改的属性的地址，这里应该表示“亮度”。
  • .B 修改的长度（B应该是字节（Byte），还有w（应该是Word，两个字节）、L（应该是Long，4个字节））。
  • =FF 要修改的值（可以改）。
