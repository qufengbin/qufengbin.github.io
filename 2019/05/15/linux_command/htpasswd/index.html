htpasswd

apache服务器创建密码认证文件

补充说明

htpasswd命令 是Apache的Web服务器内置工具，用于创建和更新储存用户名、域和用户基本认证的密码文件。

语法

htpasswd(选项)(参数)

选项

参数

  • 用户：要创建或者更新密码的用户名；
  • 密码：用户的新密码。

实例

利用htpasswd命令添加用户
