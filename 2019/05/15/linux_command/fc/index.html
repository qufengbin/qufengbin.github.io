fc

修改历史命令并执行

补充说明

fc命令 自动掉用vi编辑器修改已有历史命令，当保存时立即执行修改后的命令，也可以用来显示历史命令。fc命令编辑历史命令时，会自动调用vi编辑器。fc保存文件后，会自动执行所编辑过的命令。

选项

参数

  • 起始指令编号：指定要编辑的起始指令编号；
  • 结尾指令编号：指定要编辑的结尾指令编号。
