shell中给临时文件命名

补充说明

有时候在写Shell脚本的时候需要一些临时存储数据的才做，最适合存储临时文件数据的位置就是/tmp，因为该目录中所有的内容在系统重启后就会被清空。下面是两种方法为临时数据生成标准的文件名。

tempfile命令

tempfile命令只有在基于Debian的发行版中才默认自带，比如Ubuntu，其他发行版没有这个命令。
