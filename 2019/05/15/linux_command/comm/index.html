comm

两个文件之间的比较

补充说明

comm命令 可以用于两个文件之间的比较，它有一些选项可以用来调整输出，以便执行交集、求差、以及差集操作。

  • 交集：打印出两个文件所共有的行。
  • 求差：打印出指定文件所包含的且不相同的行。
  • 差集：打印出包含在一个文件中，但不包含在其他指定文件中的行。
