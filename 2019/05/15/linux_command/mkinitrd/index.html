mkinitrd

建立要载入ramdisk的映像文件

补充说明

mkinitrd命令 建立要载入ramdisk的映像文件，以供Linux开机时载入ramdisk。

这个是重新封包核心的命令，例如你自己修改了一个设备的驱动，如果这个驱动要加入核心级别的话，就需要对核心进行重新封包，把新加的配置编译到核心内部去！

语法

选项

参数

  • 映像文件：指定要创建的映像文件；
  • 内核版本：指定内核版本。
